## Run Locally

Clone the project

```bash
  git clone https://github.com/aks1489/Note-App.git
```

Go to the project directory

```bash
  cd Note-App
```

Install dependencies

```bash
  npm install --legacy-peer-deps
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React

**Server:** Node

**DataBase:** Google Firebase
